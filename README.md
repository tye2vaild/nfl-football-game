# NFL Football Game

## Overview
This project is a simulation of an NFL football game, allowing users to play through different scenarios, understand game mechanics, and engage in a virtual experience of American football.

## Features
- Game simulation with scoring mechanics
- Player statistics tracking
- Customizable teams
- Realistic game algorithms

## Prerequisites
- Python 3.x
- pip (Python package installer)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/tye2vaild/nfl-football-game.git
   cd nfl-football-game
   ```

2. Install necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To launch the game:
```bash
python game.py
```

## Configuration
You can modify game settings in the `config.json` file to suit your preferences.
